python语言介绍

python语言的基本特点

  • python是一种动态类型的，解释性的高级编程语言
    • 静态类型：变量类型在程序运行前确定
    • 动态类型：变量类型在程序执行时确定，可以改变
  • 语法简洁易读，编程模式非常符合人类的思维方式和习惯
  • 通用、跨平台
  • 强大的社区支持
  • 理想的“胶水语言”：具有跨语言跨平台跨技术栈的强大整合能力

python语言的优势

  • 语法简洁，同时提供丰富的数据结构和标准库
  • 易学易用，学习周期短，开发效率高
    • 语法接近自然语言
    • 虽然代码执行效率没有编译型语言高，能够节省编程时间
    • 第三方库几乎覆盖所有技术领域

python语言的缺点

  • 动态解释型语言，运行速度慢
  • 全局解释器锁(GIL)限制多线程并发
  • 源代码不能加密

python基本语法

python程序基本结构

  • 简单语句：写在一行内的语句
  • 复合语句，包含其他语句(组)的语句，需要缩进来表示语句块的层次结构(除了要求缩进的情况，其他情况不能缩进)
    • 条件语句 if elif else
      • 句头、句体
      • 句头后面都是冒号结尾
    • 循环语句 for while
    • 函数定义 def
  • 注释 #

物理行和逻辑行

  • 物理行：回车
  • 逻辑行：根据显式或隐式行拼接规则所拼接在一起的
    • 显示拼接行 \结尾
    • 隐式拼接行 圆括号、方括号、花括号内的表达式可以分成多个物理行，不必使用反斜杠
      • 隐式行拼接可含注释

标识符和保留字

python标识符

  • 用来命名变量、函数、类、对象、模块等的名称
    • 字符集：在ASCII字符集范围内，大小写字母(AZ, az)、数字(0~9)、下划线(_)
    • 首字母：不能以数字开头
    • 大小写：区分大小写
    • 保留字
    • 长度：语言对长度没有限制

python保留字

  • 关键字(保留字)
  • 软关键字
    • python新增match语句，把match, case,_定义为软关键字，仅在match语句内部作为关键字

python变量和赋值语句

变量

  • 变量：在python中，可以将变量视为指向值(数据对象)的标识符

赋值语句

  • 赋值语句：变量=表达式
  • python作为动态类型语言，变量使用前不需要声明其类型，变量的首次赋值就是其创建过程，变量的类型就是赋值给它的类型
  • 重新赋值不仅可以改变变量的值，也可以改变变量的类型
  • 在python中，’=‘不是运算符，变量=表达式是赋值语句，不是赋值表达式
    • x = (y = y + 1)用法错误
  • 赋值语句支持链式赋值
    • x = y = z = 5
  • 赋值语句支持多重赋值
    • x, y, z = 5, ‘a’, 1.5
    • 例
      • 交换两个变量的值 a, b = b, a
      • 从列表中取值
        • list = [1, 2, 3]
        • a, b, c = list
  • python3.8中，新增了表达式，操作符为:=
    • x := (y := y + 1) 合法

python中的常量

  • 常量，在程序执行中其值不会改变的变量
    • python没有提供const，没有真正意义上的常量
    • 一般默认全大写的变量不会修改

python数据类型

  • 基本数据类型
    • 整型
      • 只有一种类型int没有固定的最大值限制，实际受限于可用内存
      • 可以使用多种进制
        • 十进制 除了0以外，首位不能是0
        • 二进制0b或0B
        • 八进制0o或0O
        • 十六进制0x或0X
    • 浮点型
      • 可用十进制或科学计数法表示
      • 采用十进制表示时，可以省略小数或整数部分，只要保留小数点
      • 科学计数法使用e或E来表示幂
      • 数值范围存在限制，小树精度也存在限制
          1. 合法，等同于10.0(省略了小数部分)
        • .1 合法
        • 1E+10 合法
        • 1e10 合法
        • 3.14e-3 合法
        • 0e0
        • 1 呃0 不合法，内部不能有空格
        • 01e010 合法，浮点数解析时
